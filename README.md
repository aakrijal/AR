# Farmers Market Analysis

## Project Overview
This repository contains the code and data for my MS thesis research at Tennessee State University: 
"Examining Cultural and Economic Barriers' Impact on Frequenting Farmers' Markets."

This project examines what factors, including cultural and economic barriers on the supply and demand sides
affect how frequently different demographic groups visit farmers' markets in Tennessee

## Repository Contents

| File | Description |
|------|-------------|
| `1.zIP_CENTROID.Rmd` | Calculates geographic centroids for ZIP code level spatial analysis |
| `ZIP_DEMOGRAPHICS.Rmd` | Processes demographic data at the ZIP code level |
| `City_controls.Rmd` | Prepares city-level control variables for regression analysis |
| `final_analysis_23march.do` | Main econometric analysis in Stata — includes regression models and results |

## Methods Used
- Spatial analysis (R)
- Econometric regression analysis (Stata)
- Demographic and heterogeneity analysis
- Survey data analysis

## Software Required
- R (with tidyverse, sf packages)
- Stata
